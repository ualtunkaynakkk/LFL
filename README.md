# Penti LFL Performans Paneli

Bölge bazlı LFL satış performansını görselleştiren interaktif drill-down dashboard.

## Özellikler
- 3 Seviyeli Drill-down: OM → BM → Mağaza
- KPI Kartları: Ciro, Adet, Ziyaretçi (ZS), FBA
- Kıyaslama Rozetleri (Penti / OM / BM bazında pp farkı)
- Sıralama & Arama
- Penti Marka Renk Paleti (#E4006D)

## Kurulum
npm install && npm run dev
